# Card images go here

Drop your PNG card art into these folders so the paths match `js/cards.js`:

```
cards/
  team1/ ember.png  spark.png  wall.png  fireball.png  golem.png  dragon.png
  team2/ bubble.png frost.png  heal.png  elemental.png tidal.png  kraken.png
  team3/ herb.png   wolf.png   vine.png  eagle.png     bear.png   treant.png
```

* Each PNG **is** the whole card — the artwork, the card name, the short
  explanation, and the mana cost printed on it.
* Recommended size: a 2.5 : 3.5 ratio, e.g. **500 × 700 px**.
* The filenames just have to match the `image:` paths in `js/cards.js`.
  Change either side to whatever you like.

You don't have to add images before playing — until a PNG exists, the game
shows a neat placeholder card with the name, cost and description instead.
